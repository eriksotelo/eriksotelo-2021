portfolio website.

One of my main goals in 2021 was to rebrand in 2022. The color palette was heavily influenced by some of my favorite Holbein watercolor paints. After finalizing the color palette, my main goal became to incorporate all three of my primary areas of focus (design, development, and illustration). The result is a website that reflects not only my skills but also my personality! Can you guess some of the things I like?

Also, I would like to note that this was my first real attempt at working with Node.js. This site was built using the Next.js framework. The contact form was built using JavaScript and NodeMailer.
